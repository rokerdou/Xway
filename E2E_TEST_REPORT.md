## 数据流分析

### 客户端数据流

1. **接收本地SOCKS5请求**
   - 从本地应用接收HTTP请求
   - 解析目标地址（www.baidu.com:80）

2. **加密转发到服务端**
   - 将目标地址加密
   - 通过加密隧道发送到服务端

3. **数据中继（双向加密）**
   ```
   本地应用 ←→ 客户端 ←→ 加密 ←→ 服务端 ←→ 百度服务器
              ↓          ↑           ↓          ↑
           解密      加密          解密      加密
   ```

### 服务端数据流

1. **接收客户端连接**
   - 接受来自客户端的加密连接
   - 解密目标地址

2. **连接目标服务器**
   - 连接到百度服务器 (103.235.46.115:80)
   - 建立TCP连接

3. **数据中继（双向解密/加密）**
   ```
   客户端 ←→ 服务端 ←→ 解密 ←→ 百度服务器
     ↑          ↓           ↑          ↓
   加密        解密        加密      明文
   ```
